## Getting Started


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/tunahan1306/RastMobile.git
   ``` 
2. Install NPM packages
    ```sh
    cd Yazlab2Project3Cli
    ```
    ```sh
    npm install
    ```
    or
     ```sh
    yarn
    ```
3. Run the project
    ```sh
    ng serve
    ```
